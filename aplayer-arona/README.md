由官方開發的功能插件，主要功能:
- 提供H5音樂媒體播放器

部署:
1. 在```/src/```下創建文件夾```music```
2. 放置方式:
   - 音樂文件放置於```/src/music/${music_name}/```下
   - 封面文件放置於```/src/music/${music_name}/```下
3. 音樂文件命名方式:
   - 音樂文件必須以```track.mp3```命名，並放置於適當的文件夾下
   - 封面文件必須以```cover.png```命名，並放置於適當的文件夾下

自定義:
- 該樣式文件放置於```/src/plugins/style/aplayer-arona.css```
